JavaScript & API 100本ノック ダッシュボード

エンジニア未経験からJavaScriptの基礎とAPI連携をマスターするための、実践型学習Webアプリケーションです。

🚀 プロジェクトの概要

このアプリケーションは、JavaScriptの基本文法から、DOM操作、そして実務で不可欠なAPI連携までを100問のドリル形式で学習できるツールです。単なる問題集ではなく、ブラウザ上で実際にコードを書いて実行できるエディタ機能を搭載しています。

主な機能

100問のステップアップ形式: 基礎から応用まで体系的に学べます。

インブラウザ・エディタ: エディタを別途用意することなく、その場でJavaScriptを実行して結果を確認できます。

進捗管理ダッシュボード: Chart.jsを利用して、学習の進捗を視覚的にグラフ表示します。

自動保存機能: ローカルストレージ（LocalStorage）を利用し、ブラウザを閉じても学習記録が保持されます。

レスポンシブデザイン: Tailwind CSSにより、PCからスマホまで快適に操作可能です。

📖 学習カリキュラム

第1章：変数とデータ型 (10問)

第2章：制御構文 (15問)

第3章：関数 (15問)

第4章：配列とオブジェクト (20問)

第5章：DOM操作 (20問)

第6章：非同期処理とAPI連携 (20問)

🛠 使用技術

Frontend: HTML5, JavaScript (Vanilla JS)

Styling: Tailwind CSS (CDN)

Charts: Chart.js

Deployment: GitHub Pages

💻 使い方

問題一覧から学習したい問題を選び「練習する」をクリックします。

左側のエディタにJavaScriptのコードを記述します。

「実行する (Run)」ボタンを押すと、右側のコンソールに結果が表示されます。

正解できたら「完了にする」ボタンを押して進捗を記録しましょう！
